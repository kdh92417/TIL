## 1. ol 태그와 ul태그

- ul : unordered list  
  순서가 없는 목록 리스트
- ol : ordered list
  순서가 있는 목록 리스트

### 예시

```html
<ul>
  <li>감자</li>
  <li>고구마</li>
</ul>
<ol>
  <li>웨이트</li>
  <li>내추럴</li>
</ol>
```

> <ul>
>   <li>감자</li>
>   <li>고구마</li>
> </ul>
> <ol>
>   <li>웨이트</li>
>   <li>내추럴</li>
> </ol>
